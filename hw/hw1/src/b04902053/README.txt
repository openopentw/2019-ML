q_13.py 到 q_16.py 分別是題目第 13 題到第 16 題的做圖程式。
執行方法如下：

	usage: python q_[13 ~ 16].py [-h] [-o OUTPUT] train

	positional arguments:
	  train                 The path to the training data.

	optional arguments:
	  -h, --help            show this help message and exit
	  -o OUTPUT, --output OUTPUT
							The path to the outputing picture.

預設輸出圖片的路徑是同一層資料夾，第 13 題的圖檔名會是 q_13.png，14 ~ 16
依此類推。

需要安裝的 python package 都列在 requirements.txt 中。
